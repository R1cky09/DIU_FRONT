# Tracker de Eventos USM

Proyecto en React para buscar, filtrar e inscribirse en eventos de la Universidad Técnica Federico Santa María.

## Instalación

```bash
git clone https://github.com/R1cky09/DIU-FRONT.git
cd DIU-FRONT/client
npm install
npm start
```
